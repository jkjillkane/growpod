# Project Overview

Pre-order website for a fully automated microgreen grow chamber. The system uses an ESP32 microcontroller to control temperature, humidity, CO2 monitoring, watering, and lighting.

## Tech Stack
- Frontend: Vanilla HTML/CSS/JS
- Build: Vite
- Hosting: GitHub Pages

## Project Structure
```
/css        - Stylesheets
/js         - JavaScript modules
/images     - Static assets
/docs       - Documentation
/tasks      - Todo and lessons tracking
```

## Key Pages
- **Homepage** - Hero section, value proposition, pre-order CTA
- **How It Works** - Sensors, automation, hydroponics diagrams
- **Features** - Lighting, irrigation, climate control, monitoring
- **Sustainability** - Carbon comparison, water usage, local production
- **Market/Use Cases** - Home, restaurant, school applications
- **Technical Specs** - BOM, sensors, architecture (open hardware)
